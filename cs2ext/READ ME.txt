README
Setup

If the download is marked as untrusted, choose Continue anyway.
Download the latest offsets.rs Rust source file from CS2 Dumper.
Place offsets.rs in this folder.
If an offsets.rs file already exists, replace it with the latest version.
Do not change imgui.ini unless you know exactly what you are doing.

THE ESP WILL NOT DISPLAY IF THE OFFSETS ARE NOT UPDATED.

Safety Notes

Keep offsets current: Make sure offsets.rs is up to date before running the project. Outdated offsets can cause crashes or trigger anti-cheat detection.
Leave configuration alone: imgui.ini is included for configuration and should generally remain unchanged.
Use a disposable account: Do not use your main CS2 account. Use a separate account for any testing.
Run offline only: Always add -insecure to your CS2 Steam launch options before running this project. This helps prevent VAC modules from loading and reduces the risk of joining VAC-secured servers.
Test locally only: Use this only in local bot matches, such as by entering map de_dust2 in the developer console. Never use it in Premier, Competitive, or any live public environment.
