# README
## Setup

1. If the download is marked as untrusted, choose Continue anyway.
2. Download the latest `offsets.rs` Rust source file from CS2 Dumper.
3. Place `offsets.rs` in this folder.
4. If an `offsets.rs` file already exists, replace it with the latest version.
5. Do not change `imgui.ini` unless you know what you are doing.

## Notes

- Make sure `offsets.rs` is current before running the project.
- `imgui.ini` is included for configuration and should generally be left unchanged.

- RECOMMENDED: DO NOT USE MAIN CS2 ACC.
